# **🪖 SMART\_AI FITNESS — Панель Воина (v1.0)**

**SMART\_AI FITNESS** — это интерактивное мобильное веб\-приложение нового поколения для прокачки тела, дисциплины и личной эффективности. Это не просто фитнес-трекер. Это **RPG-путь воина**, где каждая тренировка — это боевая миссия, а ваш прогресс контролируется строгим цифровым ИИ-Командиром **ORION**.

## **⚡️ Главная концепция: Путь Воина**

Пользователь проходит путь от **Новобранца** до **Генерала**, выполняя ежедневные миссии и развивая три ключевых тактических атрибута:

* **💪 Сила** (развивается при выполнении силовых подходов)  
* **⚡️ Выносливость** (растет за счет кардио-упражнений и удержания статики)  
* **🛡 Дисциплина** (повышается за регулярность тренировок и закрытие серий)

## **🦾 Ключевые возможности проекта**

### **1\. 🤖 ИИ-Командир ORION**

Интегрированный ИИ на базе языковой модели **Google Gemini** выступает в роли вашего личного тренера. Он:

* Анализирует параметры вашего тела (вес, рост, возраст) и генерирует оптимальный 28-дневный план.  
* Общается в строгом, мотивирующем военном тоне.  
* Адаптирует нагрузку на лету (например, если вы отправляете рапорт «Я сегодня устал», Орион автоматически снижает планку на следующую миссию).

### **2\. 👁 ИИ-Контроль осанки (Computer Vision)**

Интегрированный модуль компьютерного зрения позволяет загрузить фотографию выполнения упражнения (или использовать встроенный тактический шаблон). ИИ анализирует углы суставов и геометрию спины, выдавая мгновенный критический рапорт об ошибках прямо в процессе тренировки.

### **3\. 🎯 RPG-геймификация и Путь развития**

* **Система Рангов:** Новобранец → Солдат → Сержант → Лейтенант → Капитан → Майор → Полковник → Генерал.  
* **Расчет Опыта (XP):** Выполнение упражнений дает базовый XP, закрытие полной миссии начисляет \+100 XP бонус, а удержание серии дней дает дополнительный множитель.  
* **Динамическая шкала:** Сферический неоновый индикатор на главном экране визуализирует текущий прогресс до повышения звания.

### **4\. ⏱ Иммерсивный экран тренировки**

* Интерактивный таймер отдыха и активности.  
* Голосовое сопровождение тренировки (TTS) на русском языке — Орион диктует подходы, отсчитывает время и дает команды «К бою\!».  
* Симулятор пульса в реальном времени, адаптирующийся под тип выполняемого упражнения.

### **5\. 🔐 Тактический доступ и Авторизация**

* Встроенная система регистрации и авторизации профилей воинов.  
* Интерактивная проверка надежности паролей с градацией уровней («СЛАБЫЙ», «ОПТИМАЛЬНЫЙ», «ЭЛИТНЫЙ»).  
* Диагностический терминал ОРИОН, логирующий ошибки валидации в реальном времени.

## **🎨 Стиль интерфейса: Neo Warrior UI**

Дизайн приложения разработан в соответствии с футуристической военной эстетикой:

* **Глубокий темный фон** (\#0B0E13) с наложением сетки цифрового чертежа.  
* **Электрический синий** (\#00D4FF) — цвет системных интерфейсов и калибровки.  
* **Неоновый зеленый** (\#39FF88) — цвет подтвержденных статусов, идеальной формы и готовности к бою.  
* **Янтарный** (\#FFB800) — предупреждения, тактические уведомления и средний приоритет.  
* **Glassmorphism** — размытие заднего плана на карточках для создания глубины интерфейса.

## **🛠 Технологический стек**

* **Frontend:** HTML5, CSS3, JavaScript (ES6+).  
* **Стилизация:** Tailwind CSS (с кастомной конфигурацией палитры Neo Warrior).  
* **Иконки:** Font Awesome 6\.  
* **Интеграция ИИ:** Google Gemini API (gemini-3-flash-preview для текста и зрения, структурированный JSON-вывод).  
* **Озвучка:** Web Speech API (Tactical Text-to-Speech Engine).

## **🚀 Быстрый старт**

Приложение является полностью автономным **Single-Page Application (SPA)** и не требует сложной серверной настройки.

1. **Клонируйте репозиторий:**  
   git clone \[https://github.com/your-username/smart-ai-fitness.git\](https://github.com/your-username/smart-ai-fitness.git)  
   cd smart-ai-fitness

2. **Запустите проект:**  
   Просто откройте файл index.html в любом современном браузере. Вы также можете использовать расширение **Live Server** в VS Code для локального хостинга.  
3. **Подключите ИИ (Опционально):**  
   В верхней панели разработчика (доступна на ПК) введите ваш персональный API-ключ Google Gemini.  
   *Приложение имеет умный локальный симулятор и полностью работоспособно в режиме офлайн без ключа.*
